# c01-git01 - Below commands are used for this exercise . 

1. Merging Exercise:
```
git checkout master
git merge <branch-name>
git checkout <branch-name>
git merge master

```

2. Rebasing Exercise:
```
git checkout -b <branch-name>
git commit
git checkout master
git commit
git checkout <branch_name>
git rebase master 

```

3.Revert and reset Exercise
```
git reset HEAD~1
git checkout pushed
git revert HEAD

```
